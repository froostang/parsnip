# Security policy

Parsnip is alpha software. The latest version on `main` is the only supported
development version.

Report security issues privately through
[GitHub Security Advisories](https://github.com/froostang/parsnip/security/advisories/new).
Do not open a public issue containing an exploit, secret, participant information,
private transcript, or captured answer.

Useful reports include the affected version, operating system, reproduction steps,
expected and observed behavior, and whether local buffer permissions, expiry,
integrity checks, or model-call authorization were bypassed. Use synthetic content
in reproductions whenever possible.
